# One Shared Truth Conference 2026

Conference website for [One Shared Truth 2026](https://onesharedtruth.com) at Fordham Law School, NYC.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deploy to Vercel:

```bash
npx vercel
```

Then add `onesharedtruth.com` as a custom domain in the Vercel dashboard.

## Built With

- [Next.js](https://nextjs.org/) 16
- React 19
- TypeScript
